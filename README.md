# dhis2-tools

This is the collection of tools that i use in day to day activities interacting
with dhis.

> Heads up! Its the work in progress.

## DB Credentials

Store db credentials in `bin/env.json` with the following format

```json
{
  "user": "dhis",
  "host": "localhost",
  "database": "dhis2",
  "password": "dhis",
  "port": 5432
}
```
